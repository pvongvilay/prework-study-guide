CSS

  • A margin indicates how much space we want around the outide of the element.
  • A padding indicates how much space we want around the content inside the element.
  • A selector defines the element of the attributes to which declarations will apply.
  • There are three ways to style a webpage with CSS, Inline, internal, and external style sheet.
  • Inline applys to a specifc HTML tage using a stule attribute with a CSS rule to style a particulat page element only
  • Inline is useful for quick and permanent changes but are less flexible of the 3 due to it being able to be edited indivdually when you make a change.
  • Internal contains CSS rules for the head section of the HTML file. The rules only apply to that page.
  • Single changes will apply to all the tagged page elements
  • External is the most common way to style a webpage. it allows developers to keep all their css rules in a seperate file. which makes design changes easier
